MovieFinder
===========
Needed libraries:
- [MySQL Connector/J](http://dev.mysql.com/downloads/connector/j/)
- [jCalendar](http://toedter.com/jcalendar/)

Database settings:  
Create a class named DatabaseCredentials with three Strings;
- DB_URL
- USERNAME
- PASSWORD

How to run:  
```java -jar MovieFinder.jar```
